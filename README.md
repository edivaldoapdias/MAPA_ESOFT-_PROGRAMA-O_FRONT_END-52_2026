# MAPA_ESOFT-_PROGRAMA-O_FRONT_END-52_2026
Desenvolver um site completo e responsivo em HTML5, CSS3 e JavaScript com pelo menos quatro páginas interligadas. Incluir menu de navegação, formulário de contato com validação, dois efeitos interativos em JS/jQuery e aplicar conceitos básicos de SEO no código.


# 🌐 Projeto: Cadastro Pessoal / Portfólio

## 🎯 Objetivo
Este projeto foi desenvolvido como parte da disciplina de **Desenvolvimento Front-End**, com o objetivo de criar um website completo e responsivo utilizando **HTML5, CSS3 e JavaScript**.

---

## 📂 Estrutura
- `index.html` → Página inicial
- `sobre.html` → Página sobre mim
- `portfolio.html` → Página de projetos
- `contato.html` → Página de contato
- `css/style.css` → Estilos principais
- `js/script.js` → Interatividade e validações
- `img/` → Imagens utilizadas no site

---

## ⚙️ Tecnologias Utilizadas
- **HTML5** → Estrutura semântica
- **CSS3** → Layout responsivo e estilização
- **JavaScript** → Validação de formulário e efeitos interativos
- **SEO básico** → Uso de meta tags e boas práticas

---

## 📱 Funcionalidades
- Layout responsivo que se adapta a diferentes telas
- Menu de navegação funcional
- Formulário de contato com validação em JavaScript
- Dois efeitos interativos:
  - Botão de "voltar ao topo"
  - Efeito hover em imagens do portfólio
- Aplicação de conceitos de SEO básico

---

## 🚀 Como Executar
1. Abra o projeto no **VS Code**.
2. Instale a extensão **Live Server**.
3. Clique com o botão direito em `index.html` → **Open with Live Server**.
4. O site abrirá no navegador.

---

## 📌 Desafios Enfrentados
- Ajustar o layout para diferentes tamanhos de tela.
- Implementar validações de formulário em JavaScript.
- Garantir interatividade sem comprometer a performance.

---

Considerações Finais
O desenvolvimento permitiu aplicar conceitos de HTML, CSS e JavaScript em um projeto prático. Aprendi sobre responsividade, interatividade e SEO. Como melhorias futuras, o site pode incluir integração com banco de dados e mais recursos visuais.
